AI魔法实验室——初中生人工智能启蒙平台
解决问题： 致力于打破传统AI教育的高门槛，为初中生提供从感性体验到算法逻辑平滑过渡的启蒙方案，解决枯燥难懂的教学痛点。

实现功能： 平台集成四大核心模块：“视觉魔法”利用YOLOv8模型体验真实目标检测与置信度；“近朱者赤”以交互散点图推演KNN的距离与投票机制；“抽丝剥茧”重写ID3算法核心，动态渲染并预测“是否打球”的多分支决策树；“小小大脑”用参数滑块直观模拟神经元的权重、求和与阈值激活。

应用效果： 沉浸式的“拖拽+实时图表反馈”免代码交互模式，极大激发了学生的探索欲。学生在调整参数的“游戏”中，自然内化了特征划分、置信度、权重等核心逻辑，有效培养了数据思维与人工智能素养，真正实现了前沿科技科普的寓教于乐。
